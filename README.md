# Python-graph-visualizer# Graph Plotter Using Python Turtle

This project is a **menu-driven Python program** that plots different mathematical graphs using the Turtle graphics module.

It allows the user to draw:
- Trigonometric functions
- Straight lines
- Circles
- Ellipses
- Parabolas

---

#Concepts Used
- Python Turtle Graphics
- Mathematical plotting
- Trigonometric functions
- Loops and conditional statements
- Coordinate geometry
